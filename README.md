# AdminBot - Assistente Virtual de Administração

## Sobre o projeto
O **AdminBot** é um assistente virtual desenvolvido para ajudar usuários com dúvidas sobre **administração** e **finanças básicas** de forma simples, prática e segura.

O foco do projeto é atender principalmente:
- estudantes
- pequenos empreendedores
- iniciantes na área administrativa

A proposta é oferecer explicações claras sobre conceitos importantes, apoio na organização financeira e respostas educativas, sempre respeitando limites de segurança.

---

## Objetivo
O AdminBot foi criado para:
- explicar conceitos de administração de forma simples
- ajudar na organização financeira básica
- orientar usuários com exemplos práticos
- evitar respostas fora do escopo ou informações inventadas

---

## Funcionalidades
- Explicação de conceitos administrativos
- Apoio à organização financeira
- Respostas com linguagem simples e educativa
- Tratamento de perguntas fora do escopo
- Regras de segurança para evitar alucinações
- Estrutura de resposta com explicação + exemplo + dica

---

## Público-alvo
Este agente foi pensado para pessoas que precisam de apoio inicial em temas administrativos, como:
- estudantes de administração
- pequenos negócios
- assistentes administrativos
- usuários com pouca familiaridade com finanças básicas
